== Description ==

Shoplet is a modern and responsive WooCommerce theme designed to help you create a fully functional online store. With seamless WooCommerce integration, a clean design, and customizable options, Shoplet is perfect for showcasing products and growing your eCommerce business.

== Installation ==

1. Upload the theme to your WordPress website by navigating to Appearance > Themes > Add New > Upload Theme.
2. Choose the `shoplet.zip` file and click Install.
3. Activate the theme.
